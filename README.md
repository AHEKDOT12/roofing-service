# Bishkek Roofing

🌐 Language: [English](README.en.md) | **Русский**

Лендинг для кровельного сервиса в Бишкеке. Проект показывает услуги компании, описывает порядок работы и позволяет пользователю открыть интерактивную форму предварительного расчета стоимости.

## О проекте

Приложение построено как одностраничный React-интерфейс с анимированными карточками услуг, навигацией по секциям и модальным калькулятором заявки. Пользователь может выбрать тип работ, площадь объекта, класс материалов и оставить контактные данные. Созданные заявки сохраняются локально в браузере через `localStorage` и отображаются в блоке сохраненных расчетов.

## Функциональность

- промо-страница для услуг Bishkek Roofing;
- карточки услуг с изображениями и hover-анимациями;
- раздел с этапами работы;
- модальное окно для предварительного расчета стоимости;
- выбор услуги, площади и класса надежности;
- сохранение созданных заявок в `localStorage`;
- локальная панель сохраненных заявок с возможностью удаления;
- адаптивный интерфейс для desktop и mobile.

## Услуги в интерфейсе

- ремонт кровли;
- монтаж крыши;
- фасадные работы;
- промышленный альпинизм.

## Запуск локально

**Требования:** Node.js

1. Установите зависимости:

   ```bash
   npm install
   ```

2. При необходимости создайте файл `.env.local` на основе `.env.example` и укажите переменные окружения:

   ```bash
   GEMINI_API_KEY="MY_GEMINI_API_KEY"
   APP_URL="MY_APP_URL"
   ```

3. Запустите приложение:

   ```bash
   npm run dev
   ```

4. Откройте приложение в браузере:

   ```text
   http://localhost:3000
   ```

## Данные и интеграции

Форма расчета не отправляет данные на сервер. Заявки сохраняются только в браузере пользователя через `localStorage`.

В текущей версии проекта нет описанных backend API-эндпоинтов. Переменные окружения из `.env.example` оставлены для совместимости с исходным шаблоном AI Studio.
